Cara bikin HTML

Pertama bikin folder di ZArchiver.

Lalu bikin file baru dan tambahkan .html di akhir nama file.

Langkah dasar

Buka Acode lalu buka folder yang kalian buat.

Setelah itu buka file yang ada .html nya.

Tulis html:5 lalu tekan tombol panah untuk membuat template HTML.

Ubah bagian title sesuai judul website yang kalian mau.

Di dalam body kalian bisa menulis judul dengan tag h1 sampai h6.

Untuk membuat paragraf gunakan tag p.
